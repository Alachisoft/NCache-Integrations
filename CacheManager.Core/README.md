# NCache.OSS.CacheManager.Core

**NCache Open Source implementation for CacheManager.Core** providing distributed caching and NCache Pub/Sub synchronization.

## Package Versions

| Package | Version |
|---------|---------|
| NCache.OSS.CacheManager.Core | 5.3.6.1 |
| Alachisoft.NCache.Opensource.SDK | >= 5.3.6.2 |
| CacheManager.Core | >= 3.0.0 |
| Microsoft.Extensions.Logging.Abstractions | >= 10.0.7 |

## Installation

Install via NuGet Package Manager:

```bash
dotnet add package NCache.OSS.CacheManager.Core
```

Or via Package Manager Console:

```powershell
Install-Package NCache.OSS.CacheManager.Core
```

## Prerequisites

Before using this package, ensure you have:

1. **NCache Server** – A running NCache cluster (Open Source or Enterprise)
2. **Distributed Cache** – A Replicated/Partitioned cache configured for caching
